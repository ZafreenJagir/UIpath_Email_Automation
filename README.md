# Exercise 10: Email Automation - send email

##### NAME : Zafreen J
##### REGISTER NUMBER : 212223040252


## AIM:
To send an email automatically using Gmail SMTP in UiPath Studio.

## PROCEDURE:
Open UiPath Studio and create a new process.
Install UiPath.Mail.Activities from Manage Packages.
Drag and drop the Send SMTP Mail Message activity.
Enter Gmail details:
Server: smtp.gmail.com, Port: 587, SecureConnection: StartTLS.
From & Username: your Gmail ID.
Password: Gmail App Password (from https://myaccount.google.com/apppasswords
).
To: receiver’s email, add Subject and Body.
Run the workflow to send the email automatically.
